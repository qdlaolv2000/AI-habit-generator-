📝 使用示例

示例：提高睡眠质量

步骤1：输入愿望

用户输入："我想提高睡眠质量，每天都能精力充沛地起床"

步骤2：AI生成行为建议

  • 每天晚上10点后关闭所有电子设备
  • 每天睡前读书5分钟
  • 每天晚上洗脸后涂护肤品
  • 每天睡前写下3件感恩的事情
  • 每天睡前整理明天的衣服

步骤3：筛选黄金行为

用户选择了："每天睡前读书5分钟" 作为黄金行为

步骤4：生成配方

配方：在我洗完澡之后，我会读书5分钟，为了让大脑牢记这个习惯，我会在心里说"太棒了！"。

🧠 理论基础

福格行为模型（B=MAP）

行为 = 动机 × 能力 × 提示

三要素解释：

  • 动机（Motivation）：你有多想做这件事？
  • 能力（Ability）：你是否有足够的能力或资源去做？
  • 提示（Prompt）：触发条件，提醒你执行行为

黄金行为三标准

🎯
Effective
能有效实现你的愿望
💪
Want to do
你真正想要做的
✅
Can do
你有能力做到的

📚 使用指南

1. 配置DeepSeek API

在 config.js 文件中设置你的API密钥：

// 修改以下配置 const CONFIG = { API: { apiKey: 'YOUR_DEEPSEEK_API_KEY_HERE', // 替换为你的API密钥 // ... 其他配置 } };

2. 自定义提示词

你可以在 config.js 中自定义AI生成内容的提示词，以适应不同的需求。

3. 本地运行

直接用浏览器打开 index.html 文件即可开始使用。
